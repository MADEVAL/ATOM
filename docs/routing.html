Route patterns

$app->router->get('/users/{id}', 'UserController@show');
$app->router->post('/posts/{slug}', 'PostController@create');
$app->router->put('/items/{id:\d+}', 'ItemController@update');
$app->router->patch('/users/{id}', 'UserController@patch');
$app->router->delete('/files/{hash:[a-f0-9]{32}}', 'FileController@remove');
$app->router->any('/webhook', 'WebhookController@handle');
$app->router->match(['GET', 'POST'], '/multi', 'MultiController@handle');
TokenPCREMatches
{id}[0-9]+42
{slug}[a-z0-9\-]+hello-world
{any}[^/]+anything
{all}.+path/to/file
{name:custom-pcre} - inline regex; addPattern() for named patterns

Register named patterns globally:

$app->router->addPattern('hash', '[a-f0-9]{32}');
$app->router->addPattern('uuid', '[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}');
$app->router->get('/files/{hash}', 'FileController@show');

Duplicate route names throw InvalidArgumentException. Named routes use O(1) hash map lookup.

Groups & middleware

$app->router->group('/admin', ['Auth', 'AdminOnly'], fn($r) => {
    $r->get('/dashboard', 'AdminController@index');
    $r->group('/users', ['Throttle'], fn($r) => {
        $r->get('/{id}', 'AdminController@edit');
    });
});

Route name prefixing

$app->router->namePrefix('admin.', fn($r) => {
    $r->get('/dashboard', 'AdminController@index', 'dashboard');
}); // Route name: admin.dashboard

URL generation

$app->router->get('/users/{id}', 'UserController@show', 'user.show');
$app->router->url('user.show', ['id' => 42]); // → /users/42

Health check

$app->router->health('/health', fn() => [
    'database' => $db->raw()->query('SELECT 1') !== false,
    'cache'    => true,
]);
// GET /health → 200 {"database":true,"cache":true}
// Any false → 503

Attribute routes

use Atom\Routing\Route;

class ApiController {
    #[Route('/api/items', ['GET'], 'items.list')]
    #[Route('/api/items/{id}', ['GET','DELETE'], 'items.crud')]
    public function handle(Request $request): string { ... }
}

$app->router->loadFromAttributes(__DIR__ . '/Controllers');

Attribute routes support url() and duplicate name detection.

Performance

All routes compile into one PCRE regex using branch-reset (?|...) and (*MARK). 10 000 routes = one JIT‑compiled regex call. 405 Method Not Allowed is O(1) via altRegex. URL generation is O(1) via hash map.

// Compiled: #^(?|(?<METHOD>GET)\/users\/(?<id>[0-9]+)(*:0)|...)$#xs
if (!preg_match($compiled['regex'], $method . $uri, $m)) {
    return new Response('Not Found', StatusCode::NOT_FOUND);
}
$id = (int) $m['MARK']; // → route index
$params = array_filter($m, 'is_string', ARRAY_FILTER_USE_KEY);

Cache invalidation: $app->router->clearCache(). Cache files include a framework cache version and route signature; stale or corrupted files are auto-detected and regenerated.

Cache strategy

Control where compiled routes are stored via APP_ROUTE_CACHE (Config::routeCache):

ValueBehaviour
'file' (default)Compiled to cacheDir/routes.php with var_export. Fast require on next boot.
'cache'Stored in framework Cache abstraction. Useful for distributed/shared caches.

Same principle applies to APP_VIEW_CACHE for compiled templates.
