JSON body auto‑parse

// Content-Type: application/json  →  auto‑parsed from php://input
// Supports CONTENT_TYPE and HTTP_CONTENT_TYPE server keys
// POST {"name":"Alice"}  →  $req->input('name') = 'Alice'
// Post‑max‑size respected via ini_parse_quantity()

Method spoofing

// POST with _method=PUT  →  $req->method === 'PUT'
// Works only from POST; only GET POST PUT PATCH DELETE HEAD allowed
// Whitespace trimmed; array values fall back to POST

Bearer token

// Authorization: Bearer abc123  →  $req->bearer === 'abc123'
// Multiline tokens rejected (regex .+ doesn't cross newlines)

File uploads

$file = $req->file('avatar');
if ($file->ok) {
    $file->move('/uploads/' . $file->name);
}
$file->size;   // bytes
$file->type;   // MIME type
$file->ext;    // extension via PCRE

Nested upload arrays return UploadedFile::empty(). Null‑bytes and path‑traversal blocked in move().

Response

// Factories
Response::html('<h1>Hello</h1>');
Response::json(['key' => 'value'], pretty: true);
Response::text('plain text');
Response::noContent();                                    // 204
Response::redirect('/login');                               // 302
Response::redirect('/new', StatusCode::PERMANENT_REDIRECT);   // 308

// Chaining — immutable, returns new instances
(new Response('body', StatusCode::CREATED))
    ->withHeader('X-Custom', 'val')
    ->withCookie('session', $token, 7200)
    ->withCookie('pref', 'dark', ['ttl' => 86400, 'samesite' => 'Strict'])
    ->withCache(3600)
    ->send(isHttps: true);

\r\n stripped from header keys and values. Cookies default: httponly: true, samesite: Lax (invalid values fall back to Lax). Cookie name/value cannot be overridden via options array. javascript:, data:, vbscript: redirects blocked.

Response introspection

$response->getContent();      // string — response body
$response->getStatusCode();  // int — HTTP status (200, 404, etc.)
$response->getHeader('Content-Type'); // string — header value or ''

// Immutable status change
$ok = $response->withStatus(StatusCode::OK);  // returns new instance

Controllers

use Atom\Http\{Request, Response, StatusCode};

class UserController {
    public function show(string $id, Request $request): Response {
        return Response::json(findUser($id));
    }
    public function __invoke(): string {
        return 'Hello World'; // auto‑wrapped to HTML
    }
}

Parameters auto‑populated from route {params}. Add a Request $request parameter for full request access.

Session

// Lazy singleton — no session_start() until first use
$session = $app->container->make(Session::class);

$session = new Session(['cookie_samesite' => 'Strict']);
$session->set('user_id', 42);
$session->get('user_id');
$session->has('user_id');
$session->remove('user_id');
$session->flash('success', 'Saved!');
$session->regenerate();

// CSRF — rotated after successful validation
$token = $session->csrfToken();
$login = $session->csrfToken('login'); // per‑form isolation
$session->validateCsrf($login, 'login');
