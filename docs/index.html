Quick start

use Atom\{Application, Config};

$app = new Application(new Config(
    debug: true,
    viewsDir: __DIR__ . '/../views',
    cacheDir: __DIR__ . '/../storage/cache',
));

$app->router->get('/', 'HomeController@index');
$app->router->group('/api', ['auth'], fn($r) => {
    $r->get('/users/{id}', 'UserController@show', 'user.show');
    $r->post('/users', 'UserController@create');
});
$app->run();

TokenPCREMatches
{id}[0-9]+42
{slug}[a-z0-9\-]+hello-world
{any}[^/]+anything
{all}.+path/to/file
{name:custom-pcre} - inline regex; addPattern() for named patterns

Duplicate route names throw InvalidArgumentException. Named routes use O(1) hash map lookup.

Performance

All routes compile into one PCRE regex using branch-reset (?|...) and (*MARK). 10 000 routes = one JIT‑compiled regex call. 405 Method Not Allowed is O(1) via altRegex. URL generation is O(1) via hash map.

// Compiled: #^(?|(?<METHOD>GET)\/users\/(?<id>[0-9]+)(*:0)|...)$#xs
if (!preg_match($compiled['regex'], $method . $uri, $m)) {
    return new Response('Not Found', StatusCode::NOT_FOUND);
}
$id = (int) $m['MARK']; // → route index
$params = array_filter($m, 'is_string', ARRAY_FILTER_USE_KEY);

Cache invalidation: $app->router->clearCache(). Corrupted cache files are auto-detected and regenerated.

Response

// Factories
Response::html('<h1>Hello</h1>');
Response::json(['key' => 'value'], pretty: true);
Response::text('plain text');
Response::noContent();                                    // 204
Response::redirect('/login');                               // 302
Response::redirect('/new', StatusCode::PERMANENT_REDIRECT);   // 308

// Chaining — immutable, returns new instances
(new Response('body', StatusCode::CREATED))
    ->withHeader('X-Custom', 'val')
    ->withCookie('session', $token, 7200)
    ->withCookie('pref', 'dark', ['ttl' => 86400, 'samesite' => 'Strict'])
    ->withCache(3600)
    ->send(isHttps: true);

\r\n stripped from header keys and values. Cookies default: httponly: true, samesite: Lax (invalid values fall back to Lax). Cookie name/value cannot be overridden via options array. javascript:, data:, vbscript: redirects blocked.

Validation — DTO definition

use Atom\Validation\{Required, Email, Min, Max, Regex, Integer, In, Ip, FloatVal, Boolean, Uuid, Each};

final class CreateUser {
    #[Required] public string $name = '';
    #[Required] #[Email(unicode: true)] public string $email = '';
    #[Min(8)] #[Max(128)] public string $password = '';
    #[Regex('/^[a-z0-9_]{3,20}$/')] public string $username = '';
    #[Integer(min: 0, max: 150)] public int $age = 0;
    #[In(['admin', 'user', 'guest'])] public string $role = 'user';
    #[Ip(noPrivate: true)] public string $publicIp = '';
    #[FloatVal(min: 0.0, max: 100.0)] public float $score = 0.0;
    #[Boolean] public bool $active = true;
    #[Uuid] public string $id = '';
}

Nested array validation

final class OrderItem {
    #[Required] public string $sku = '';
    #[Min(1)] public int $qty = 0;
}

final class CreateOrder {
    #[Required] public string $customer = '';
    #[Each(OrderItem::class)] public array $items = [];
}
// Errors returned as [0].sku, [1].qty, etc.

Attribute reference

AttributeFilter / Logic
#[Required]Non‑null, non‑empty string, non‑[]
#[Email(unicode: true)]FILTER_VALIDATE_EMAIL (RFC 822) + UNICODE flag

#[Min(N)]String length ≥ N or numeric value ≥ N
#[Max(N)]String length ≤ N or numeric value ≤ N

#[Between(min, max)]Numeric or string length range
#[In(['a','b'])]Strict comparison against list
#[Url]FILTER_VALIDATE_URL (RFC 2396)
#[Nullable]Skip when null
#[Confirmed]Must match {field}_confirmation
#[Ip(onlyV4, onlyV6, noReserved, noPrivate, global)]FILTER_VALIDATE_IP + flags

#[Mac]FILTER_VALIDATE_MAC
#[FloatVal(min, max)]FILTER_VALIDATE_FLOAT + optional range
#[Boolean]FILTER_VALIDATE_BOOL (1/0/true/false/on/off/yes/no)

All attributes accept custom messages: #[Required('Name is required')].

Middleware pipeline

class Auth implements MiddlewareInterface {
    public function handle(Request $req, Closure $next): Response {
        if (!$req->bearer) {
            return new Response('Unauthorized', StatusCode::UNAUTHORIZED);
        }
        return $next($req);
    }
}

// Object, string (DI-resolved), or Closure
$r->get('/secure', 'Controller@action', '', ['Auth']);
$r->get('/debug', 'DebugController@index', '', [
    fn(Request $req, Closure $next) => $next($req)->withHeader('X-Debug', '1')
]);
