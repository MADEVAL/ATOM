WebSocket ⚡

First-class WebSocket support · RFC 6455 · Real-time rooms · Zero dependencies

RFC 6455Frame codecEvent loopRoomsBroadcast stream_selectNon-blocking

Introduction

Atom includes a complete WebSocket server with RFC 6455 frame encoding/decoding, non-blocking event loop via stream_select, room/channel management, and broadcasting. The entire implementation is ~250 lines of code across two classes.

No external dependencies, no separate process, no PHP extensions beyond what's already compiled. The WebSocket server runs in the same PHP process.

Connection

Handshake (SHA-1 + Base64), frame encode/decode, masking, ping/pong, close

Server

Event loop, connection accept, routing with {params}, rooms, broadcast

Handlers

onOpen / onMessage / onClose via event string

Rooms

join, leave, sendToRoom — per-room broadcasting

Quick start

use Atom\{Application, Config};
use Atom\WebSocket\Connection;

$app = new Application(new Config(debug: true));

// Register a WebSocket handler
$app->ws('/chat', function(Connection $conn, $data, $event) {
    if ($event === 'open') {
        $conn->sendJson(['type' => 'welcome', 'id' => $conn->id()]);
    }
    if ($event === 'message') {
        $conn->sendJson(['echo' => $data]);
    }
});

// Start server: php atom ws:serve

Handlers

The handler receives 4 arguments: Connection, data, event, and params.

Route parameters

$app->ws('/chat/{room}', function(Connection $conn, $data, $event, $params) {
    // $params['room'] = 'general' for ws://host/chat/general
    $server = $app->wsServer();
    $server->join($params['room'], $conn);
});

Rooms

Rooms are in-memory collections of connections. Join, leave, and send to specific rooms.

$server = $app->wsServer();

// Connection management
$server->join('room-42', $conn);
$server->leave('room-42', $conn);

// Send to everyone in a room
$server->sendJsonToRoom('room-42', [
    'type' => 'message',
    'from' => $conn->id(),
    'text' => $data['text'],
]);

// Send raw text to all connections  
$server->broadcast('Server shutting down');

// Send JSON to all connections
$server->broadcastJson(['type' => 'announcement']);

Complete chat room example

$app->ws('/chat/{room}', function(Connection $conn, $data, $event, $params) use ($app) {
    $server = $app->wsServer();
    $room = $params['room'];

    switch ($event) {
        case 'open':
            $server->join($room, $conn);
            $conn->data['room'] = $room;
            $server->sendJsonToRoom($room, ['type' => 'join', 'id' => $conn->id()]);
            break;
        case 'message':
            $server->sendJsonToRoom($room, ['type' => 'msg', 'text' => $data['text']]);
            break;
        case 'close':
            $server->leave($room, $conn);
            $server->sendJsonToRoom($room, ['type' => 'leave', 'id' => $conn->id()]);
            break;
    }
});

Protocol

Full RFC 6455 implementation:

  • Opcode support: text (0x1), binary (0x2), close (0x8), ping (0x9), pong (0xA)
  • Payload lengths: 7-bit (≤125), 16-bit (≤65535), 64-bit
  • Masking: Client→Server frames unmasked automatically
  • Auto-pong: Ping frames automatically answered with pong
  • Close handshake: Close frame echoed back, connection marked closed
