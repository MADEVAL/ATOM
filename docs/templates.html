Template syntax & filters

{# comment #}
{{ user.name | e }}
{{ html | raw }}
{{ title | upper | raw }}
{{ count | default(0) }}
{{ data | json }}
{{ items.0 }}
escapeeupperlowertrimlengthnl2brjsondefaultraw

default filter shows fallback for null, false, '', and []. Nested braces {{ func({a: 1}) }} supported. Auto‑escaping applied unless raw|escape|e|json|nl2br used.

Control flow

{% if user.admin %}
  <p>Admin</p>
{% elseif user.active %}
  <p>Active</p>
{% else %}
  <p>Inactive</p>
{% endif %}

{% for item in items %}
  <li>{{ item | e }}</li>
{% endfor %}

{% for key, val in data %}
  {{ key }}: {{ val }}
{% endfor %}

{% set total = price * qty %}
{% include "partials/nav.twig" %}

For‑loop variables are cleaned up after endfor. Shadowed variables (same name as outer context) are saved and restored. Unclosed tags throw RuntimeException.

{{ title }}                    {# "My Page" #}
{% for title in items %}
  [{{ title }}]                {# iterated #}
{% endfor %}
{{ title }}                    {# "My Page" — restored #}

Inheritance

{# layout.twig #}
<html>
<head><title>{% block title %}Default{% endblock %}</title></head>
<body>{% block body %}{% endblock %}</body>
</html>

{# page.twig #}
{% extends "layout.twig" %}
{% block title %}{{ page.title | e }}{% endblock %}
{% block body %}<p>{{ page.content | raw }}</p>{% endblock %}
